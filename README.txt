# Unique Username Registration System

A simple Java application that prevents duplicate usernames using the `Set` data structure.

This project demonstrates how `HashSet` provides efficient O(1) lookup for detecting duplicate entries.

## Features

- Register new users
- Prevent duplicate usernames
- Display all registered users
- Command-line interface

## Technologies

- Java
- HashSet (Java Collections Framework)

## Example Output

Enter username: brandon
User registered successfully.

Enter username: brandon
Username already exists.

Enter username: luke
User registered successfully.
